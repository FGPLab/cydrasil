# Notes about refactoring database and description language choices

## Naming requirements for Newick
------------------------------------------------------------------------------------------------------------------------
RAxML-ng name requirements for newick compatability

NOTE: Following symbols are not allowed in taxa names to ensure Newick compatibility:
NOTE: " " (space), ";" (semicolon), ":" (colon), "," (comma), "()" (parentheses), "'" (quote).
NOTE: Please either correct the names manually, or use the reduced alignment file
NOTE: generated by RAxML-NG (see above).

## Description language choices for certain parameters
------------------------------------------------------------------------------------------------------------------------
- Identical sequence with ORGANISM_NAME after masking with SSU-Align
- 16S sequence trimmed from operon
- Multiple 16S copies
- AKA ORGANISM_NAME; Also known as
- FKA ORGANISM_NAME; Formerly known as
